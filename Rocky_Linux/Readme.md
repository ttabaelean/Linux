

# 🐧 Linux 서버 관리자 과정 (Rocky Linux 기반)
본 과정은 기업 환경에서 널리 사용되는 Enterprise Linux의 표준인 Rocky Linux를 활용하여, 가상화 환경 구축부터 서버관리자를 위한 리눅스 시스템 관리, 3-Tier 아키텍처 기반의 서비스 운영까지 시스템 엔지니어에게 필요한 필수 역량을 체계적으로 습득하는 과정입니다.

<br>


### 🎯 학습 목표
가상화 환경 구축: VMware 환경에 Rocky Linux를 설치하고 실무용 원격 접속 환경을 완벽하게 구축합니다.
시스템 제어: 리눅스 핵심 명령어와 쉘 환경을 숙달하여 설정, 권한 및 프로세스를 효율적으로 제어합니다.
인프라 관리: 네트워크 구성 원리와 파일시스템을 이해하고 스토리지 확장 및 로그 분석 대응 능력을 갖춥니다.
3-Tier 서비스 구축: Web, WAS, DB 서버를 직접 설치하고 상호 연동하여 실제 서비스 아키텍처를 완성합니다.

<br>
