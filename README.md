# 🐧 Linux Lab Repository
이 저장소는 리눅스 서버 운영 및 인프라 구축 실습을 위한 가이드와 설정 문서를 관리하는 공간입니다. 실제 강의에서 다루는 실무 예제들을 중심으로 구성되어 있습니다.

<br>

### 📂 주요 실습 내용
현재 Rocky_Linux 디렉토리를 중심으로 다음과 같은 실습 자료가 포함되어 있습니다.
- 시스템 기초: 환경 설정, 기본 명령어, 프로세스 및 사용자 권한 관리
- 네트워크 & 스토리지: IP 설정, 파티션 생성(fdisk), 파일시스템 포맷 및 마운트
- 서비스 운영: DNF 패키지 관리, systemd 서비스 제어, 로그 관리 및 분석
- 인프라 구축: Web-WAS-DB 연동을 통한 3-Tier 서비스 아키텍처 실습
